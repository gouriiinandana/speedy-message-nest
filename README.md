# Chat Application


This is a simple chat application where multiple users can sign up, log in, and chat with each other in real time.

## Features
- User signup and login
- Multiple users can chat
- Real-time messaging
- Secure authentication

## How to Run
1. Clone the repository
2. Install dependencies
3. Start the server
4. Open the application in the browser

## Future Improvements
- Private chats
- User profiles
- File sharing

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
